# Steam Roguelike Market Analysis

A data analysis project exploring the growth of roguelike games on Steam using Python, Pandas, and Matplotlib.

## Project Goals

- Analyze yearly roguelike game releases
- Compare roguelike growth with other Steam tags
- Explore Steam genre trends
- Analyze tag popularity and market direction

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Findings

- Roguelike releases increased rapidly after 2020
- Survival remains the largest Steam tag
- Deckbuilding and roguelike genres show strong growth trends

## Dataset

Steam Games Dataset from Kaggle.
Due to file size limitations, the original dataset is not included in this repository.
